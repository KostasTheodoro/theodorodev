Στα πρώτα στάδια, ήταν μια απλή ιστοσελίδα σε Next.js και React με Tailwind CSS
και Headless UI, χωρίς βάση δεδομένων λόγω του ασφυκτικού χρονοδιαγράμματος που
είχαμε για να βγει live. Το έργο συνεχίζει να εξελίσσεται το 2026: έχουμε πλέον
ενσωματώσει Supabase και Sanity CMS ώστε να αυτοματοποιείται πλήρως η προσθήκη
και αποθήκευση νέων προϊόντων. Ξεκίνησε ως συνεργασία με έναν άλλον developer
στα πρώτα στάδια, και έκτοτε είναι ένα ατομικό project, το οποίο συνεχίζω ενεργά
να βελτιώνω.
